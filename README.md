# Adhell3-En-to-Cn  
Adhell3的app汉化文件  
使用原作者的[编译脚本](https://gitlab.com/fusionjack/adhell3-scripts)时,执行一次Adhell3 build后将本文件（string.xml）拷贝至adhell3-master\app\src\main\res\values-zh-rCN目录下（不存在则新建），再编译。
